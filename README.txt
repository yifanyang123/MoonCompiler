To compile the moon code, just change the attribute mySrc in driver

Then the compiler will generate corresponding outAST tree, derivation process, lextokens, SymbleTable, different type of errors in moonCode folder.

And Finally it will generate *.m file, which is the low level langauge of moon virtual machine
To test the final result, run moon.exe with command line.
